## 🚀 Getting Started  

### Clone the repository  
```bash
git clone https://github.com/Zaidshaikh2811/ZaidShaikh.git
cd ZaidShaikh
````

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
